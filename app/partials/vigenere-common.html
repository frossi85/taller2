Vigenere - Encriptación

Para el cifrado se utiliza el abecedario ingles, el cual solo incluye letras de la A a la Z sin incluir la ñ. De esta manera tenemos un alfabeto de encriptación de 26 letras.

Para encriptar el texto ingresado se necesita una clave de encriptación en el alfabeto antes mecionado, esta clave se repite hasta que tenga el mismi largo que el texto a encriptar y luego se procede a utilizar la siguiente formula para encriptar caracter a caracter.

Yi = (Xi + Zi) mod n

Donde:

  • Yi = posicion en alfabeto de caracter encriptado
  • Xi = posicion en alfabeto de caracter a encriptar
  • Zi = posicion en alfabeto de caracter de la clave utilizado para encriptar
  • n = cantidad de letras del alfabeto usado (en este caso 26)
